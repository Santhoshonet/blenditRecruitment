Tech Specialist Job Description

We are looking for someone that can figure out any software application. Basically we are looking for someone that would love nothing better than to get their hands on a new piece of software and figure out how to get it to work and make sure it continues to work properly. We also are looking for someone that can improve the way we deploy and manage applications. This might be anything from an off the shelf product to a script you cook up.

Ohh! You will also need to learn how to script, so that means that you will have to get pretty comfortable with the command line. It is ok if you have not worked with the command line before, but you will have to get up to speed fast!

To start off with you will primarily be working with Microsoft SharePoint and Project Server

Responsibilities

  • Software installation - To start off with it will be primarily Microsoft SharePoint and Project Server
  • Configuration
  • Liaise with clients to help them solve problems they might have with the software we deploy
